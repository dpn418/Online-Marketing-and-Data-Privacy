About this website

This website is an IS 117 assignment for NJIT professor Keith Williams. Each of the chapters is writing done by me to summarize topics that I have learned about.

Assignment Details

The objective of this project is to create a website that explains how online data is collected, how it can be used, and what you can do to remain anonymous online. You need to include a section in the website about why you should care and you can include your own thoughts in this section. You need to create a website with around 10 pages

You need to design the visuals and write in a style that would be appealing to one of the following 3 demographics:

  1. Teens
  2. College students / Young Adults
  3. Your grandparents

The website should include a carousel rotator on the home page. You are required to use bootstrap for the entire site, you will lose significant points for not doing this. Implement F-layout for the content pages, Z-layout for your homepage. Also include 5-10 images in your website and the images should be from pulbic domain.
